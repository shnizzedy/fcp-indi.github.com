Atlas
`````

    An *atlas* provides a guide to the location of anatomical features in a coordinate space.

    -- :cite:`Poldrack, et al., 2011 <PoMN11>, Mejia 2015 <Meji15>` p. 55

An atlas provides a defined coordinate space, a `template </glossary/template>`_ for aligning images, and labels for regions of interest.
