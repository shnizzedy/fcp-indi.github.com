Data¶

Where are your data?¶

On a local drive¶

Are they in BIDS format?

  • Yes

    • What is the path to the BIDS directory?
      • [populate -v/-B mappings with this to /data etc., add to run command]

  • No

    • How to generate a data configuration?
    1. Use the CLI to generate a template file
    2. Fill in the template
      1. Set dataFormat to Custom
      2. Fill in anatomicalTempate and functionalTemplate with the absolute path (/path/to/data_dir), following the examples in the template comments to specify the directory structure
      3. Fill in outputSubjectListLocation and subjectListName
      4. Fill in any other relevant keys in the template file
    3. Use the CLI to generate configuration files
    4. In the container run command, include the flag –data_config_file /path/to/data_config.yml (replacing “/path/to/data_config.yml` to the absolute path to your configuration file generated in the previous step). bids_dir is still required but not used; the configuration file includes the absolute data paths
