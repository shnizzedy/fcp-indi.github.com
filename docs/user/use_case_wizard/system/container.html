How to pull C-PAC image?¶

Docker

Docker¶

To pull the latest version of C-PAC from Docker Hub:

docker pull fcpindi/c-pac:latest

To pull a different version, please specify the image tag. For example, you can pull fcpindi/c-pac:release-v1.5.0 to use C-PAC version 1.5.0; pull fcpindi/c-pac:nightly to use C-PAC development version (under development, not stable).

Singularity

Singularity¶

To pull the latest version of C-PAC from Singularity Hub:

singularity pull --name C-PAC.sif shub://FCP-INDI/C-PAC
